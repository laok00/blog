shadowsocks-config

2016-05-29

This is a short reference of shadowsocks setting on debian client.

reference:
https://wiki.archlinux.org/index.php/Shadowsocks
The arch wiki is perfect, simple and useful.

configure:

/etc/shadowsocks/config.json
---
{
"server":"remote-shadowsocks-server-ip-addr",
"server_port":443,
"local_address":"127.0.0.1",
"local_port":1080,
"password":"your-passwd",
"timeout":300,
"method":"aes-256-cfb",
"fast_open":false,
"workers":1
}

client

From the command line
$ sslocal -c /etc/shadowsocks/config.json

$ echo alias ssl='sslocal -c /etc/shadowsocks/config.json' >> .bashrc
